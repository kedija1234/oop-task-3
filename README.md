# Simple Programmer System

This is a beginner Java console project.

## About
This project takes user input and stores information about a programmer such as name, age, and favorite programming language.

## Concepts Used
- Encapsulation
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- ArrayList
- Scanner
- Exception Handling

## How It Works
The user enters details, the program creates a programmer object, and then displays the information.
